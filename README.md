## How can I edit this code?

There are several ways of editing your portfolio.

**Use your preferred IDE**

Clone this repo and work locally with your own IDE. The only requirement is having Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

Follow these steps:

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd ai-architect-portfolio

# Step 3: Install dependencies
npm i

# Step 4: Start the development server
npm run dev
```

**Edit files directly in GitHub**

- Navigate to the desired file(s).
- Click the "Edit" button (pencil icon) at the top right.
- Make your changes and commit.

**Use GitHub Codespaces**

- Navigate to your repository main page.
- Click the "Code" button (green) near the top right.
- Select "Codespaces" tab and click "New codespace".
- Edit files and commit/push when done.
